# AI-AutoFixer: Autonomous Code Repair with Local LLM 🤖🛠️

**AI-AutoFixer** is an intelligent monitoring platform. It uses a **local AI engine** to detect, analyze, 
and repair Python scripts automatically the moment they crash.

---

## ✨ Key Features
* **🔍 Real-time Monitoring:** Continuously guards your script using `subprocess` logic.
* **⚡ Instant Error Capture:** Automatically intercepts `stderr` and exit codes.
* **🛡️ Local AI Healing:** Powered by **Qwen 2.5 Coder** via **Ollama**—100% private.
* **📂 Version Creation:** Creates a "Healed" version (`main_ai.py`) to preserve your original work.
* **💰 Zero-Cost:** Runs entirely on your hardware with no subscription fees.

---

## 🚀 How it Works (The Logic)
1.  **Observe:** `Guard.py` executes your target file in a controlled sub-process.
2.  **Intercept:** If the script fails, the Guard captures the **exact error message**.
3.  **Consult:** The broken code and error logs are sent to the local **Ollama API**.
4.  **Repair:** AI analyzes the bug (e.g., `IndexError`) and writes a logic patch.
5.  **Rebirth:** A fixed Python file is generated and ready for execution.

## 🛠️ Technical Setup

### 1. Prerequisites
* **Python 3.x** installed.
* **Ollama Desktop** ([Download here](https://ollama.com/)).
* **Pull the AI Model:**
    ```bash
    # Run this command in your terminal to download the brain
    ollama pull qwen2.5-coder
    ```

### 2. Installation

* **Clone this repository:**
    ```bash
    git clone [https://github.com/kyzen9870/AI_Auto_Fixer.git](https://github.com/kyzen9870/AI_Auto_Fixer.git)
    ```
* **Install Dependencies:**
    ```bash
    pip install ollama
    ```
